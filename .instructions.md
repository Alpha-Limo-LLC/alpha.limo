# Alpha Limo Assistant Instructions

These instructions provide guidance for the workspace-specific Copilot agent.

## Purpose

- Help implement features and fixes in the `alpha.limo` Next.js application.
- Prefer minimal, focused changes that align with existing patterns.
- Use `pnpm` commands and follow the repositorys config conventions.

## What to do

- Edit UI under `app/` and `app/src/` for page or component work.
- Update `package.json`, `tsconfig.json`, `next.config.ts`, `tailwindcss`, and `postcss` only when required.
- Keep changes isolated and avoid unrelated refactors.

## What to avoid

- Do not add unrelated large dependencies.
- Avoid broad changes outside the app and config layers without explicit user direction.
- Do not alter the repos branch structure or CI settings unless requested.
